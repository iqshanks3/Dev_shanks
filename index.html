دليل تثبيت Kali NetHunter في Termux

الخطوة 1: تحديث حزم تيرمكس وإعطاء صلاحية التخزين

افتح تطبيق Termux واكتب الأمر التالي لتحديث النظام بالكامل والموافقة على صلاحيات الوصول للملفات:

termux-setup-storage && apt update && apt upgrade -y
✓ تم النسخ

الخطوة 2: تثبيت أداة الـ Wget

نحتاج لتثبيت أداة wget لتحميل السكربت الرسمي لـ NetHunter من موقع كالي:

pkg install wget -y
✓ تم النسخ

الخطوة 3: تحميل سكربت التثبيت الرسمي

قم بسحب ملف تثبيت نيت هنتر مباشرة من سيرفرات Kali Linux الرسمية:

wget -O install-nethunter-termux https://gitlab.com/kalilinux/nethunter/build-scripts/kali-nethunter-project/-/raw/master/nethunter-rootless/install-nethunter-termux
✓ تم النسخ

الخطوة 4: إعطاء صلاحيات التنفيذ للسكربت

تعديل صلاحيات الملف المحمل ليصبح قابلاً للتشغيل داخل بيئة الأندرويد:

chmod +x install-nethunter-termux
✓ تم النسخ

الخطوة 5: بدء تشغيل التثبيت

شغل السكربت الآن واتركه يقوم بتحميل وتثبيت واجهة كالي لينكس بالكامل (سيستغرق بعض الوقت بناءً على سرعة الإنترنت لديك):

./install-nethunter-termux
✓ تم النسخ

الخطوة 6: تشغيل نظام Kali Linux

بعد اكتمال التثبيت، يمكنك الدخول إلى النظام في أي وقت عبر كتابة الأمر الحركي:

nethunter
✓ تم النسخ
